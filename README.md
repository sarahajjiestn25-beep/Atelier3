## 📌 Description

Ce TP a pour objectif de pratiquer les bases du langage **PHP** à travers plusieurs exercices.
Les exercices portent principalement sur :

* la manipulation des **variables et constantes**
* l'utilisation des **instructions de sortie**
* les **conditions (if, else, switch)**
* les **boucles (for, while, do while)**

---

## 🧪 Exercices réalisés

Les exercices réalisés commencent à partir de la **page 6** du cahier d’exercices.

### Partie : Contrôles de flux et boucles

* Exercice 1 : Condition IF simple
* Exercice 2 : Correction d’un script avec IF ELSE
* Exercice 3 : Différence entre IF et SWITCH
* Exercice 4 : Calculer un prix avec IF et ELSEIF
* Exercice 5 : Afficher les nombres pairs avec FOR
* Exercice 6 : Correction d’une boucle FOR
* Exercice 7 : Différence entre WHILE et DO WHILE
* Exercice 8 : Affichage des nombres avec WHILE et DO WHILE
* Exercice 9 : Boucle FOR imbriquée (table de multiplication)

## ▶️ Exécution

1. Placer les fichiers dans le dossier **htdocs** de XAMPP.
2. Démarrer **Apache** dans le panneau de contrôle XAMPP.
3. Ouvrir le navigateur et accéder au fichier PHP.

Exemple :

```
http://localhost/nom_du_dossier/fichier.php
```

## 🎯 Objectif

L'objectif de ce TP est de comprendre les concepts fondamentaux de PHP et de s’entraîner à utiliser les structures de contrôle et les boucles pour résoudre différents problèmes.
